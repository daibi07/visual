## 项目内容

1. 展示温度、UVB、露点(dew point)和全球新增确诊人数的关系

2. 展示GDP、人口、地域面积和总计确诊人数的关系

3. 展示国内出行强度和国内确诊人数的关系

4. 从全球地图的尺度展示各个地域新增确诊人数人数、总计确诊人数、死亡人数

5. 从国内地图的尺度展示出行强度
